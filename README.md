# 🖼️ Art Gallery — ASP.NET MVC

A dynamic online art gallery built with ASP.NET MVC and SQL Server. Artists can showcase their work and visitors can browse, search, and interact with artworks.

## ✨ Features
- 🎨 Browse & search artwork collections
- 👤 User registration & authentication
- 🖼️ Artist portfolio pages
- 🗄️ SQL Server database backend
- 📱 Responsive MVC architecture
